# What is testdoc?

Testdoc is an experiment of mine to see if Antora could be used effectively to create
documentation for a complex software application.  The results of my experimentation is
a documentation website of an excellent open source application I use called Not1mm. 
See it here.  It took me less than 8 hours to convert the Tex file of the original 
documentation to a website using Asciidoctor and Antora.

## Who am I

I am retired from a career as an application developer, project manager, IT 
director and college instructor in Computer Science.  I have always wanted 
to contribute to an open source software project but while I was still employed
and raising a family I just never found the time.  I also felt that my 
programming skills weren't up to the task for most software systems.  

Then I retired.  Perhaps I could put my skills to work on assisting with
documentation?  Documentation is the bane of every software developer and the most
difficult component to keep current with the code.

### Background

During my 15 years as a computer science college instructor, one of my major pain
points was updating my course materials such as my syllabi and handouts for each
new semester.  I had to update a number of dates, links and other items that
changed every semester.  I discovered Asciidoctor,  an implementation of 
the markdown language Asciidoc, which I think is markdown on steroids.

### Why markdown?

I liked using an ASCII document source so if I wanted to, I could version my
documents in Github, something that is not easy to do with Microsoft Word, Pages or 
a typical word processor file.

### Why Asciidoctor?

Markdown is an excellent document creation language, but it didn't support tables
well and in academia we love tables.  Asciidoctor allowed me to imbed macros into
the source code of a document to calculate due dates for assignments, making my 
course materials easy to modify for the coming term.  Yes, I could have used TeX.  I 
guess it just wasn't for me.  Just a personal thing.

### What is Antora?

Antora was written by the creators of Asciidoc as a framework and set of tools 
that can be used to create a portable website of documentation.
